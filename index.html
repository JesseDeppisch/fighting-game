What is this project all about?

This project is all about testing our Java coding progress, and to pose a challenge to us. The game is a 2D fighting game, and we're utilizing the Slick2D library to make it a bit easier. We hope for it to be similar to other fighting games, such as street fighter, but this project is really about learning, not creating a beautiful end project (that'd be nice, too). So, keep this in mind.
